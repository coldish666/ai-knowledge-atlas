# AI Roadmap

## 主干路线

数学基础 → Python 与数据处理 → 经典机器学习 → 深度学习 → Transformer / 核心模型家族 → 大模型 → RAG → Agent → 工程化。

## 层级结构

- L0 数学与计算基础
- L1 编程与数据处理
- L2 经典机器学习
- L3 深度学习
- L4 核心模型家族
- L5 大模型与生成式 AI
- L6 RAG 与知识库
- L7 AI Agent
- L8 AI 工程化
- L9 前沿方向

## 枝干方向

- AI for Science：PINN、Neural ODE、FNO。
- 视觉 / 遥感 AI：CNN、ResNet、U-Net、ViT、分割。
- 机器人 / 强化学习：具身智能、世界模型、Agent 规划。
- 大模型应用工程：Prompt、RAG、Agentic RAG、评测。
- 推理部署：ONNX、TensorRT、量化、模型服务。

## 图谱结构

图谱边来自 `prerequisites`、`next_topics` 和 `related_topics`。例如梯度下降连接反向传播，Embedding 连接 Tokenizer、Self-Attention 和 RAG Embedding。
