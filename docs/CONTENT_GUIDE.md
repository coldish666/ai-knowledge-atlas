# Content Guide

## 如何添加新知识点

1. 在 `backend/app/seed/seed_knowledge.py` 的 `TOPICS` 中添加一条主题记录。
2. 设置稳定 `slug`、标题、层级、分类、难度、标签、摘要和数学表达。
3. 如需特殊关系，在 `EXPLICIT_RELATED` 中补充相关 slug。
4. 删除本地旧数据库 `data/knowledge_atlas.db`，重启后端重新 seed。
5. 运行 `pytest` 和前端构建。

## 内容模板

每个知识点应回答：

- 它是什么？
- 直觉上怎么理解？
- 为什么重要？
- 有什么数学表达？
- 最小代码怎么写？
- 能用在哪里？
- 初学者最容易误解什么？
- 需要先学什么？
- 学完之后该连到哪里？
